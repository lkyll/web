# Personal Portfolio – SE112 Course Project

This is a **static personal website (online CV / portfolio)** built with pure HTML and CSS.  
It meets all requirements for Part A of the SE112 project:

- ✅ 4 pages (Home, About, Projects, Contact)
- ✅ Semantic HTML5 (header, nav, main, section, article, footer)
- ✅ Pure CSS with **Flexbox** and **CSS Grid**
- ✅ Fully responsive (mobile friendly)
- ✅ No JavaScript frameworks, no CSS frameworks like Bootstrap
